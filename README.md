# BanHammer (Mace Ban & Strike Engine) 🔨

**BanHammer** is a legendary punishment plugin designed for **Paper 1.21.1+** Minecraft servers. It introduces a powerful Ban Hammer (using the 1.21 Mace item) that instantly strikes, applies custom potion effects, and bans players upon hit with visual lightning & explosion effects.

---

## Core Features

- 1.21 Mace Integration: Custom model data support (\`mace_banhammertexture.png\`).
- Configurable Hit Effects: Lightning strikes, visual explosions, wither sounds, and particle bursts.
- Configurable Potion Effects: Apply Blindness, Slowness, Wither, or Nausea to targets on hit.
- Adventure MiniMessage: Fully customizable rich-text color formatting and broadcast messages.

---

## Commands & Permissions

- \`/banhammer give <player>\` - Gives the legendary Ban Hammer (Permission: \`banhammer.admin\`)
